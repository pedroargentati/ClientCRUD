# ClientCRUD

- Projeto criado para estudos.
- Contém todas operações para manupulação de dados de clientes CRUD ( CREATE, READ, UPDATE e DELETE ).


# Tecnologias
- Java 17;
- Servlets;
- TomCat 9.0;
- JSP ( Java Server Pages );
- HTML5;
- CSS3.
